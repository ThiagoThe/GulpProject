- GulpProject

É um Trabalho final da conclusão do curso de Gulp no ano de 2018.
Foi atualizado alguns arquivos em 2023. 

- Instalar e rodar o projeto

Rodar o GulpProject em sua máquina local é uma tarefa simples.

- Dependências

Você precisa ter uma dependência instalada:

Node.js LTS v16 (ou qualquer versão superior)

- Dependências locais

Então após baixar o repositório, não se esqueça de instalar as dependências locais do projeto:

npm install

- Rodar o projeto

É só abrir a página Index.html no localhost da sua máquina.
